# Acknowledged destructive migrations

# Checked by scripts/check-migration-safety.sh (runs in CI).

# EA-SYS migrates BEFORE the blue-green swap, so the old slot serves live
# traffic against the new schema for a few seconds. A DROP / RENAME / SET NOT
# NULL / ALTER TYPE on a column the old code still touches will 500 real users
# — and a code rollback re-runs `migrate deploy`, so it cannot undo it.
#
# Prefer expand/contract: add now, drop in a later deploy once no code reads it.
#
